Usando links

Aula 01 - Links Externos

Links externos são links que estão fora do nosso domínio.
Você pode acessar o meu GitHub.

Para envelopar/envolver: Ctrl + Shift + P + Enter + TAG de interesse (só para relembrar)

A tag 'a' significa âncora.

Para que seja criada uma âncora com link que abrirá uma nova aba, coloque o argumento "target="_blank" rel="external", depois do link.

Você pode acessar o meu GitHub.




Aula 02 - Links Internos

Acesse a segunda página

Para links internos, defina qual será a página anterior com rel="prev" e a página seguinte com rel="next"

CHAT GPT: Os atributos rel="next" e rel="prev" no elemento 'a' são utilizados para definir a relação entre o documento atual e o documento vinculado, facilitando a navegação sequencial entre páginas, especialmente em contextos como documentos paginados, séries de artigos, ou passos de um tutorial.

A terceira página, eu criei dentro de uma pasta entitulada "noticias"

Para links internos da própria página, não é necessário colocar nada de comandos. Porém pode ser inserido em target="_self", mas não é obrigatório.

./ = própria pasta

../ = página anterior

Links para downloads

Utilize o comando "download" para determinar qual arquivo dentro da pasta será baixado.

Utilize type="application/..." e altere as reticencias de acordo com o tipo de arquivo que será baixado.
