Capítulo 11 - Mídias

Aula 01 - Imagens dinâmicas (responsivas)

Abra o GIMP; clique em arquivo > nova; altere o tamanho conforme você for utilizar em dispositivos diferentes.

Em opções avançadas coloque resolução de 80px; para celular coloque tamanho da imagem de 300px, para tablet 700px e para TVs/PCs 1000px.

Pintamos as imagens, uma de cada cor para melhor diferenciar e salvamos.

Aula 02 - Elemento Picture (continuação da aula 01)

A tag picture é semelhante com a img, mas ela te permite criar diversas sources, isto é, fontes. Com ela é possível definir o max-width, sem utilizar CSS, apenas HTML.

Aula 03 - Áudios

Dica para utilizar músicas de fundo do youtube:

Abra o YoutubeStudio; vá em bilbioteca de áudio; se escolher uma sem atribuição necessária não é preciso referenciar, mas se escolher com atribuição, deve-se colocar os direitos autorais (as linhas que vêm abaixo do áudio, para referenciar). No meu caso, escolhi uma que não tem atribuição necessária.

Reproduzindo Áudio

A maneira mais simples é colocar a tag audio; adicionar o autoplay para tocar automaticamente, e controls para aparecer a caixinha de som.

Outra forma é apagar os argumentos de dentro da tag audio, e colocar a tag source para adicionar os áudios.

Aula 04 - Vídeo

Os formatos que apresentam melhor compatibilidade são: mp4/m4v, ogv, webm;

Handbrake: é um software que é utilizado para converter seus vídeos, caso precise inserir um vídeo interno no seu site. Ele não converte pra ogg, mas os principais mesmo são mp4 e webm. Guanabara usou o video-online-convert.com para converter para ogv.

Este vídeo está hospedado no meu próprio servidor.

Aula 05 - Vídeos de Hospedagem Externa

A ideia da aula 04 pode gerar um custo muito alto pro cliente, já que teria que colocar diversos tipos de formatos de vídeo, e as vezes com o grande acesso ele teria que pagar um valor considerável para manter o site para todos os usuários (?); portanto, agora vamos ver uma forma mais útil de fazer tudo isso pegar.

Portanto, o que faremos é adicionar vídeos de hospedagem de vídeos; no caso, será pelo Youtube.

Quando dá play neste segundo vídeo, o usuário não está consumindo dados do cliente, mas sim do YouTube.

O youtube só não contempla quando você quer deixar o vídeo privado apenas para um nicho de pessoas; pra isso deve-se procurar outros servidores de hospedagem de vídeos. O que o GUanabara usa é o Vimeo.

É possível também inserir vídeos presentes no vimeo. Ele não entrega tanto quanto os vídeos do youtube, porém a qualidade dele é muito melhor, e nos dá muitas opções para incorporar.
