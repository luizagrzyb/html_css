Explicando as pseudo-classes

O 'visited' vai gerar alterações caso você queira que o link que já foi acessado anteriormente tenha uma aparencia diferente aos demais.

O 'active' gera alterações quando o link está selecionado.

Portanto, pseudo-classes são palavras que definem estados de elementos de dentro do HTML

Explicando pseudo-elementos

Os elementos funcionam para os ids e para as classes (pelo que entendi são os before e after).

Resumão dos seletores personalizados

# = id
. = class
: = pseudo-class
> = children (tipo herança, acho)
