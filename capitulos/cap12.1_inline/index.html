Capítulo 12 - Iniciando aprendizado em CSS

Aula 01 - Estilo CSS inline (menos recomendado)

Utilizando esta forma de chamar o CSS, você estará inserindo os estilos das CSS dentro do HTML. Para isso, deve-se colocar "style" dentro das tags.

Desta forma, você estará poluindo o seu HTML.

Quando utiliza font-size e, em vez de colocar em pixels, coloca em em, você está definindo a partir do tamanho base da fonte. Por exemplo, se você padronizou 20px para todos os tópicos, e depois colocou 0.7em, ele irá reduzir em 0.7x o tamanho da fonte.
