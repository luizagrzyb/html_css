EX.: HTML básico

Quando você tem diversos tipos de texto iguais (como vários h1, h2, p, etc) e gostaria de alterar uma(s) dessa(s), você pode dar a elas uma identificação — um 'id'.

ANOTAÇÕES IMPORTANTES DE AULAS PASSADAS:
Agora, se você quiser que mostre o que estiver entre os símbolos de maior e menor, você deve colocar '< ;' e '> ;', sem espaço (abra no vs code, para ver como deve ser colocado de fato). Por exemplo:

<p id="paragrafo01">

Com isso, vá ao CSS e escreva, por exemplo: 'p#paragrafo01{}', pois desta forma você está dizendo que, dentro dos diversos parágrafos 'p', há um que se chama/é identificado como 'paragrafo01' e é apenas ele que você deseja realizar alterações.

Você também pode colocar apenas '#parágrafo01{}'. Isso ajuda caso você queira chamar outras partes do texto com a mesma identificação, independente se é parágrafo, título, etc, pois gerará uma alteração em todos. Porém, se quiser alterar especificamente o parágrafo dito como parágrafo01, e em outro lugar você colocou uma id com o mesmo nome, deve-se colocar sim o tipo antes do #.

Por exemplo: criarei uma id de 'especial' para todos os h2 de HTML.


HTML: id; CSS: #;
HTML: class; CSS: .;

Enquanto eu fazia esta aula, pensei que para padronizar algumas das tags (como parágrafos específicos, títulos diferentes mas que queria que tivesse a mesma caracteristica) pudesse utilizar o <p#id:"nomeDaIdentificacao">, porém está errado!!! devemos utilizar neste caso as CLASSES
