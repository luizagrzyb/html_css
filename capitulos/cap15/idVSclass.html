Capítulo 15 - diferenças entre id e class

Enquanto eu fazia esta aula, pensei que para padronizar algumas das tags (como parágrafos específicos, títulos diferentes mas que queria que tivesse a mesma caracteristica) pudesse utilizar o <p#id:"nomeDaIdentificacao">, porém está errado!!! devemos utilizar neste caso as CLASSES

Então, id é para um único elemento; class para vários.

DICA DO DIA MENINAS: Não dê nome de id/class pela forma, mas sim por funcionalidade

('verde' x 'basico')

Outra dica, beninex: para alterar (apagar) mais de uma tag com o mesmo nome, pressione a tecla 'alt' e clique nas tags q quer apagar.

Quando você tem dois elementos e deseja aplicar o mesmo estilo para ambos,
NÃO SE UTILIZA id IGUAIS! Utiliza, na verdade as CLASS!!!!!

SPAN no html é muito importante! Pois dentro de tag <span> você cita a classe criada em CSS que deseja inserir as configurações desta. Por exemplo:

<span class="importante">

ANOTAÇÕES IMPORTANTES DE AULAS PASSADAS:
Agora, se você quiser que mostre o que estiver entre os símbolos de maior e menor, você deve colocar '< ;' e '> ;', sem espaço (abra no vs code, para ver como deve ser colocado de fato). Por exemplo:

HTML: id; CSS: #;
HTML: class; CSS: .;

Uma outra informação importantíssima

é que cada tag pode fazer parte de mais de uma classe e/ou id (mas o id se sobrepõe!!!!!). Tem como colocar style dentro de span também, bem bacana, fiz nesta frase anterior. Mas voltando ao que foi dito no começo, trago como exemplo:

<span class="importante avancado">
