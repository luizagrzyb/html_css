Principais formatações

Negrito/Destaque

Nessa frase temos um pedaço em negrito usando a tag 'b'. Esta tag é uma tag não-semântica, diferentemente da maioria das outras tags dentro de HTML5 (conhecido também como HTML semântico).
Ou seja, ela não tem significado, só tem forma.

Nesta frade, tenho um termo em destaque usando a tag 'strong', que é semântico.



Itálico/ênfase

Nesta frase temos um termo em itálico usando a tag 'i' (não semântico).


Obs: Caso você queira colocar alguma informação dentro de uma tag, selecione-a completamente, clique em ctrl + shift + 'p', busque por "emmet envolver com abreviação" e escreva a tag desejada.
