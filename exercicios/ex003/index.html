Capítulo 06 - Imagens em HTML: testando carga e imagens


Abaixo você vai ver uma imagem que está na mesma pasta.

Para carregar, digite a tag img; coloque entre as aspas depois de src e pressione crtl + espaço e selecione a imagem de interesse. A parte de alt serve para inserir textos alternativos.

Aqui nós redimensionamos os tamanhos e resoluções das imagens.

Caso altere o nome do arquivo, você deve alterar no código também.

Podemos inserir imagens dentro de subpastas.

Também podemos carregar imagens externas. Apenas copie o endereço da imagem que estiver online.
