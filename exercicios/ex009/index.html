Três listas possíveis em HTML

Ordenadas

A tag 'ol' gera listas ordenadas (order list). Dentro dela haverá a tag 'li', que significa 'list iten'. A tag 'li' não precisa ser fechada, é opcional em HTML5.
É possível colocar um parâmetro chamado 'type', com opções 1, i, I, a ou A. Se quiser iniciar a lista a partir de um número, escreva o parâmtro 'start'.

  1. acordar;
  2. tomar café;
  3. ligar para joão;
  4. escovar os dentes;
  5. ir para faculdade;
  6. jantar;
  7. dormir.

Não ordenadas

São aquelas que não precisa enumerar. Para isso utilize a tag 'ul' (unorder list). Também é possível personalizar o tipo, com 'type' sendo os argumentos possíveis 'circle', 'disc', 'square'.

  • café;
  • feijão;
  • sucrilhos;
  • manjericão.
