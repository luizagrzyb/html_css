Citações simples

Para colocar o sentido de uma citação, coloque a tag 'q' de quote.
Como diria o pai de um amigo: o computador é um burro muito rápido.

Citações completas

Para colocar uma citação completa, utilize a tag 'blockquote':

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Obcaecati ut molestias quia, quis aliquid repudiandae aut. Modi doloremque amet vel velit, culpa porro maiores, fuga optio, sunt voluptas atque nam.

Dentro dessa tag é possível colocar um argumento chamado 'cite' em que você poderá anexar o link de onde retirou o texto. Aqui, não será gerada uma citação visível, nem link de onde foi retirada a frase, mas é uma forma de possibilitar, por meio da busca, que o computador reconheça e faça uma associação entre o texto citado e o texto que está citando, pois compreende que há uma relação entre eles.
